# AI 本地知识库问答助手 - 产品使用手册

\*\*

## 文档说明

- 适用版本：基础版 / 进阶版 / 桌面客户端版

* 核心定位：本地部署的私有化知识库工具，支持多格式文档上传、AI 精准问答，数据不上云

- 适用人群：学生、职场人、开发者、小微企业

## 一、产品概述

### 1. 核心功能

功能模块

核心价值

多格式文档上传

支持 PDF/TXT/Markdown/Word 格式，自动解析文本内容

AI 精准问答

基于上传文档提问，AI 仅依据文档内容回答，避免 “胡说八道”

流式输出

类似 ChatGPT 逐字显示回答，提升交互体验

历史对话管理

保存提问与回答记录，支持查看、清空

知识库重置

一键清空当前知识库，重新上传文档

进阶功能（可选）

多知识库管理、模型切换、对话导出、文档检索高亮、密码登录

### 2. 产品优势

- 隐私安全：所有数据（文档、对话、向量）存储在本地，不上传任何云端服务器

* 轻量化：无需复杂部署，个人电脑即可运行，包体积＜100MB

- 易用性：操作简单，上传文档→提问→得到答案，3 步搞定

* 可扩展：全栈源码支持二次开发，进阶版支持功能定制

## 二、快速开始（3 分钟上手）

### 1. 环境要求

运行环境

最低配置

操作系统

Windows 10+/MacOS 12+/Linux（Ubuntu 20.04+）

### 2. 启动步骤 // 目前没有

#### （1）桌面客户端版（推荐，免部署）

1. 解压安装包（Windows 为 .exe 文件，Mac 为 .dmg 文件）；

1) 双击启动程序（Windows 双击 AI知识库助手.exe，Mac 拖拽到应用程序文件夹后打开）；

1. 程序启动后自动打开浏览器（或内置窗口），进入问答页面；

1) 配置 AI 模型（见「三、详细操作 - 4. 模型配置」），即可开始使用。

#### （2）源码版（需简单部署）

1. 解压源码压缩包，进入根目录；

1) 复制 backend/.env.example 为 backend/.env，填写 AI 模型 API 密钥（如 OpenAI / 通义千问）；

1. 复制 frontend/.env.local.example 为 frontend/.env.local（默认无需修改）；

1) 安装依赖：根目录执行命令 npm install（需提前安装 Node.js）；

1. 启动服务：根目录执行命令 npm run dev；

1) 启动成功后，访问 <http://localhost:3001> 进入问答页面。

## 三、详细操作指南

### 1. 文档上传与管理

#### （1）上传文档

- 方式 1：点击左侧面板「点击上传」按钮，选择本地文件（支持多选）；

* 方式 2：直接拖拽文件到上传区域（支持批量拖拽）；

- 支持格式：PDF、TXT、Markdown（.md）、Word（.docx）；

* 限制说明：单文件大小≤50MB，批量上传建议不超过 10 个文件（避免内存占用过高）。

#### （2）上传反馈

- 上传中：显示进度条（0%\~100%），提示 “上传中...”；

* 解析中：上传完成后自动解析文本，提示 “解析中，请稍候...”；

- 成功：提示 “文件上传并解析成功”，左侧面板显示已上传文件列表（含文件名、大小、上传时间）；

* 失败：提示失败原因（如 “不支持的文件类型”“文件损坏”“大小超过限制”），请根据提示调整后重新上传。

#### （3）知识库重置

- 操作路径：左侧面板「知识库操作」→ 点击「重置知识库」按钮；

* 功能效果：清空当前所有上传的文档和向量库数据，需重新上传文档才能继续提问；

- 注意事项：重置后数据不可恢复，请谨慎操作。

### 2. AI 问答交互

#### （1）提问操作

- 在右侧聊天区输入框中输入问题（如 “文档中提到的 XX 方案有哪些步骤？”）；

* 支持回车发送或点击「发送」按钮；

- 提问技巧：

* <br />

- 明确问题核心（如 “2024 年 AI 行业趋势有哪些？” 而非 “文档讲了什么？”）；

* <br />

- 多轮对话时可关联上下文（如 “上一个问题的答案再详细说明下”）；

* <br />

- 如需精准定位，可指定文档（如 “根据《产品需求文档.pdf》，用户登录流程是什么？”）。

#### （2）回答查看

- 流式输出：AI 回答将逐字显示，无需等待全部生成；

* 取消回答：回答生成过程中可点击「取消」按钮，终止当前回答；

- 检索高亮（进阶版）：回答中会标注引用的文档片段和页码（PDF 格式），点击可跳转查看原文。

#### （3）历史对话管理

- 查看历史：右侧聊天区上方自动显示所有历史对话（按时间倒序排列）；

* 清空对话：点击右侧面板「清空历史对话」按钮，清除所有历史记录（不可恢复）；

- 导出对话（进阶版）：点击「导出」按钮，选择 TXT/MD 格式，将历史对话保存到本地。

### 3. 进阶功能操作（仅进阶版 / 桌面客户端版）

#### （1）多知识库管理

- 新建知识库：点击顶部导航栏「知识库」→「新建」，输入知识库名称（如 “前端面试”“论文资料”）；

* 切换知识库：在知识库下拉列表中选择已创建的知识库，当前上传的文档和对话仅关联该知识库；

- 删除知识库：点击知识库名称右侧「删除」按钮，需确认后删除（关联的文档和对话将一并删除）。

#### （2）模型切换

- 操作路径：顶部导航栏「设置」→「模型配置」；

* 支持模型：GPT-3.5/GPT-4、通义千问、DeepSeek、本地模型（如 Llama 2）；

- 配置步骤：

1. <br />

1) 选择目标模型；

1. <br />

1) 填写对应模型的 API 密钥（从模型官方平台申请，如 OpenAI 官网、阿里云通义千问平台）；

1. <br />

1) 点击「保存配置」，立即生效。

#### （3）权限控制（密码登录）

- 开启密码：「设置」→「权限设置」，勾选「启用密码登录」，设置登录密码；

* 登录验证：下次启动产品时，需输入密码才能进入问答页面；

- 修改 / 重置密码：在权限设置页面可修改密码，忘记密码需删除本地配置文件（路径：backend/storage/config.json）。

- 费用说明：API 调用会产生费用（如 GPT-3.5 按 tokens 计费），建议关注平台计费规则，避免超额。

## 四、常见问题（FAQ）

### 1. 文档上传相关

#### Q1：上传文件提示 “不支持的文件类型”？

A1：当前仅支持 PDF/TXT/Markdown/Word（.docx）格式，如需支持其他格式（如 Excel），可购买定制版。

#### Q2：上传 PDF 后，AI 回答显示 “无相关信息”？

A2：可能是 PDF 为扫描件（图片格式），当前版本不支持 OCR 识别，建议转换为可编辑的 PDF 或 TXT 格式后重新上传。

#### Q3：上传大文件（如 50MB）时卡住？

A3：建议关闭其他占用内存的程序，或拆分文件后分批上传；进阶版支持批量上传优化，可考虑升级。

### 2. AI 问答相关

#### Q1：AI 回答与文档内容不一致？

A1：可能是提问不够明确，或文档中无相关信息，建议：① 细化提问（如指定章节、关键词）；② 检查文档是否上传成功；③ 重置知识库后重新上传文档。

#### Q2：流式输出卡顿或中断？

A2：① 网络不稳定（非本地模型）：检查网络连接，配置稳定代理；② 内存不足：关闭其他程序释放内存；③ 模型 API 限流：更换模型或稍后再试。

#### Q3：历史对话丢失？

A3：历史对话存储在本地文件（backend/storage/history/history.json），若该文件被删除或修改，会导致对话丢失，建议定期备份该文件。

### 3. 部署与启动相关

#### Q1：源码版启动提示 “Node.js 版本过低”？

A1：需安装 Node.js v16+ 版本，下载地址：<https://nodejs.org/>，安装后重启终端重新执行命令。

#### Q2：桌面客户端启动后无法打开页面？

A2：① 检查是否已有其他程序占用 3000/3001 端口，关闭占用程序后重新启动；② 若仍无法打开，可联系技术支持（见下方联系方式）。

#### Q3：离线使用时，AI 无法回答？

A3：需配置本地模型（如 Llama 2），并在「设置→模型配置」中选择本地模型，基础版不支持本地模型，需升级至进阶版。

### 4. 其他问题

#### Q1：如何导出历史对话？

A1：进阶版支持导出为 TXT/MD 格式，操作路径：右侧聊天区→「导出」按钮→选择格式→保存到本地。

#### Q2：如何备份知识库？

A2：知识库数据存储在 backend/storage/chroma 目录，复制该目录即可备份，恢复时覆盖该目录即可。

#### Q3：需要二次开发，如何获取技术支持？

A3：购买源码版后，可加入客户群获取基础技术支持，如需深度定制（如对接企业微信、新增功能），可购买定制开发服务。
